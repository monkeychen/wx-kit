# wx-kit 文库素材创作编排

## Overview

把 wx-kit 文库的文章变成创作素材，编排「选料 → 选题 → 写作」一条链。**这是个薄编排层**：选料靠 wx-kit 的 CLI/导出文件，写作靠 `khazix-writer`（你的笔调在那儿），本 skill 只负责把素材拉过来、提选题、在两个关键点停下等你拍板。

**核心原则:人在环中。** AI 收集素材、提选题候选、出初稿;**选题和定稿权在你**。两个检查点不可绕过——不要从素材一路冲到成品。

## When to Use

- 想拿文库里若干篇文章（或某天订阅自动下载的）当素材，写一篇新稿。
- 已在文库 GUI 多选并「导出为素材」，手里有一个 `exports/*.json`。
- 想先从一批素材里看看「能提炼出哪些选题方向」再决定写哪个。

不适用：还没下载文章（先用 wx-kit 下）、手上没素材纯从零写（直接 khazix-writer）。

## 流程（四步，两个检查点）

### 1. 取素材

两条路，任选其一：

- **GUI 已导出**：用户在文库多选 →「导出为素材」会写出 `<库根>/exports/<时间戳>.json`。读最新的那个清单文件。
- **CLI 现取**：跑 wx-kit 的 `library export`（命令与打包/开发环境路径见 `agent/README.md`），选料器：
  - `--ids a,b,c`（按文章 id）
  - `--since YYYY-MM-DD`（按下载日期，「今天自动订阅下载的」用今天日期）
  - `--account <公众号名>`（按号，昵称包含匹配）
  - `--all`（全库；无选料器时必须显式给）

清单是 JSON：`{ ok, count, articles:[{ id,title,account,author,publishTime,sourceUrl,dir,contentPath }] }`。**正文不在清单里**——按需读每篇的 `contentPath`（即 `dir/content.md`）拿全文。

如果用户没说选哪些，先问清楚（哪个号/哪几篇/哪个时间段），别自作主张导全库。

### 2. 选题（轻量内联，借横纵交叉的思路）

读完选中文章的正文后，提炼 **3–5 个选题方向**，每个给：

- **角度**：一句话说清这篇要从什么切口写。
- **理由**：为什么这个角度成立/有意思（可借横纵思路——这批素材里的「时间线变化」或「彼此对照/张力」往往就是选题）。
- **HKR 自评**：Happy（有趣/有悬念）、Knowledge（有信息量）、Resonance（戳情绪）——标各项强弱，帮你快速判断。

> 这步是「从已有素材里提选题」，**不调 hv-analysis**（那是重量级单主题联网深研→出 PDF 的工具，错位）。若某个选题你想要联网深研支撑，见下方「旁路」。

### 🛑 检查点 1：你拍板选题

把候选列给用户，**停下等他选/改**。用户可能选一个、合并两个、或给新角度+结构。**在用户拍板前，绝不进入写作。**

### 3. 写作

把「选定的选题（角度 + 用户给的任何结构/要点）+ 选中文章正文」作为素材，**交给 `khazix-writer` skill 出初稿**（它自带 HKR 选题质检与卡兹克笔调；本 skill 不重复造写作能力）。

### 🛑 检查点 2：你审定

把初稿给用户，**停下等他审/改**。按反馈迭代，直到他满意。审阅/校验（事实核查、查重）是可选的后续，本 skill 不强求一版到位。

## 旁路：需要联网深研时

若某个选题需要超出现有素材的事实/竞品/历史支撑，**单独**对该主题跑 `hv-analysis`（产出 PDF 研究报告），再把结论并入素材交给 khazix-writer。这是可选加深，不在默认流程里。

## 边界（勿越）

- 本 skill 只经 wx-kit 的 **CLI / 导出的 `exports/*.json`** 取数据，**绝不直接读 `library.json` 或 import wx-kit 代码**——换任何下载器/agent 都按同一契约接入。
- 不在本 skill 里重写选题/写作能力:选题轻量内联、写作全权交 khazix-writer。
- 素材用途:选题/研究/启发 + 人工主导创作;**不洗稿、不自动转发**。

## Common Mistakes

- **跳过检查点直接出成品**：最常见的错。两个 🛑 必须停。
- **自作主张 `--all` 导全库**：用户没指定就先问，别一股脑全导。
- **把清单里的 `contentPath` 当正文**：那是路径，要去读文件。
- **为选题去跑 hv-analysis**：错位且慢；选题轻量内联即可。
