# v0.2.1 · 2026-06-09

安全补丁版。功能与 v0.2.0 完全一致，仅升级运行时与构建工具链，清掉 v0.2.0 发布后 GitHub Dependabot 报的全部漏洞提示。

> **未签名**首次打开仍需手动放行，见 README「30 秒上手(下载安装包)」。

---

## 🔒 依赖安全升级

- **electron 31.7.7 → 42.3.3**：修复一批 use-after-free、ASAR 完整性绕过、service-worker IPC 欺骗、clipboard 崩溃等 CVE。
- **electron-builder 24.13.3 → 26.15.2**：其捆绑的 `tar`（→7.5.11+）、`glob`（→10.5.0+）升到修复版，清掉打包工具链的路径穿越类告警。
- **vite 5.4 → 6.4.3**、**vitest 2 → 3.2.6**（v0.2.0 收尾时已升）：修 dev-server 路径穿越 / UI server 文件读取类告警。

> Dependabot 提示从 v0.2.0 时的 26 项清零。

## ✅ 验证

- 119 单测 + tsc + lint + 24 步 Playwright e2e 全绿（electron 42）。
- mac dmg(arm64+x64) + win nsis exe 重新出包；electron 二进制与构建工具链均走 npmmirror 国内镜像下载。
- **真实启动打包后的 .app + 触发 cheerio 解析路径，零 console/page 错误** —— `undici` external 不变量在 electron 42 仍成立（不会因 `node:sqlite` 缺失而崩）。

## 📦 安装包

- macOS (Intel):       `wx-kit-0.2.1.dmg`

- 依赖下载改走国内镜像优先策略（见 `AGENTS.md` 与 `docs/plans/2026-06-09-deps-audit.md`）：`ELECTRON_MIRROR` / `ELECTRON_BUILDER_BINARIES_MIRROR` 指向 npmmirror，镜像域名加 `no_proxy` 直连，代理仅作国外站不可达时兜底。
