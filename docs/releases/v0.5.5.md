# wx-kit v0.5.5 — 文库目录化导航 + 按关键词筛选下载（2026-07-18）

## 新增与改进

**文库导航:首屏即目录（M23）**——治「滚动好久才看到下面的文章」:

- 分组视图**默认收起**,打开文库先看到「公众号 · N 篇」的目录,一屏尽览,点开哪个看哪个;展开状态记忆(跨页面/跨重启保持),新公众号默认收起;工具栏一键「全部展开/全部收起」。
- **粘性组头**:滚动时当前组头钉在顶部(卡片/列表视图皆是),随时可点收起、直接落到下一组;列表视图列头同钉。
- **回到顶部**悬浮按钮。
- 性能实测背书:1000 篇假文库下目录态打开 ~170ms;卡片全展开后滚动曾降到 ~23fps,`content-visibility: auto` 修至 ~52fps——千篇量级流畅,无需虚拟滚动。

**按公众号下载支持关键词筛选（M24,issue #1）**:

- 抓取配置新增「关键词筛选」:下拉选**仅下载含 / 排除含**(互斥),关键词逗号分隔、按标题匹配(不区分大小写),留空不筛。
- 过滤发生在「列出 → 下载」之间,**零额外请求、不加重频控**;只在指定抓取范围内筛,不为凑数翻页。
- CLI 同步支持 `crawl --include <kw,kw>` / `--exclude <kw,kw>`(可组合,exclude 优先),JSON 输出含 `filteredOut`(被过滤篇数)。
