# wx-kit v0.6.0 — Agent 自动化闭环:命令行安装、登录态迁移、专属 skill(2026-07-19)

## 新增

**命令行安装通道(M26)**——agent/用户一条命令装好,不再手动下 dmg:

```sh
# macOS(Homebrew)
brew install --cask monkeychen/wx-kit/wx-kit
xattr -cr /Applications/wx-kit.app     # 未签名 app 必须清 quarantine,否则 CLI 也会被 Gatekeeper 卡住

# macOS / Linux(npm,Node 20+;国内先设 ELECTRON_MIRROR)
npm install -g @simiam/wx-kit
```

**登录态跨机器迁移(M27)**——headless/服务器环境无法扫码的正解:

- `wx-kit session export [-o <file>]`:导出登录态(文件权限 0600,等同登录凭证,用后即删);
- `wx-kit session import <file>`:结构校验 → 导入 → **立即真探测有效性**(失效如实报 `valid:false`)。
- 工作流:mac 扫码 login → export → scp → 服务器 import → agent 全自动 crawl/订阅检查。

**wx-kit 使用 skill(M28)**——`agent/wx-kit-skill/`:agent 的安装与使用教程(检测未装即自动安装、登录态分场景处理、全部 CLI 原子能力速查、组合任务范例、频控纪律),样例逐条实测;与 `wx-kit-compose`(创作编排)互补。
