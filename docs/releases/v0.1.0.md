# v0.1.0 · 2026-06-07

第一阶段首发:**URL 下载 + 公众号批量爬取 + 库内阅读 + CLI**,带可分发的 macOS / Windows 安装包。

> **未签名**首次打开需手动放行,见 README「下载与安装」一节。macOS 后续会做签名+公证。

---

## ✨ 新增功能

### M1 · 核心层 + CLI `download`

- `src/core/` UI 无关纯逻辑:解析/导出/库索引/队列全可单测。
- `npx electron . download --url <u> --formats md,html,pdf,meta --out <dir>`,支持批量 URL 与文件。
- 五种导出格式:封面、Markdown、网页、PDF、元信息。
- 文章库自动建索引(`library.json`),断点续传、按 `articleId` 去重。

### M2 · GUI

- 暖色编辑杂志风:刊头横导航、封面卡片书架、editorial 阅读版面、友好格式选择。
- 四个页面:URL 下载(实时进度)、书架(搜索/删除/在文件夹中显示)、内置阅读器(md/html,带 `wxfile://` 协议与防穿越)、设置。
- Playwright Electron 端到端测试覆盖完整 GUI 流程。

### M3 · 扫码登录 + 公众号批量爬取(CLI)

- `npx electron . login` 扫码 → 持久化 session 到 userData(本地,不同步上云)。
- `auth-status / search <name> / crawl <name> (--count N | --from/--to)` / `library list`。
- 串行 + 随机延迟(翻页 1–3s、下载 2–5s);命中 `ret=200013` 频控自动指数退避;`200040` 失效返回 `AUTH_REQUIRED`。
- CLI 输出纯 JSON:`stdout` 走数据,`stderr` 走进度,退出码 `0/1/2`。

### M3.5 · 批量爬取 GUI 页

- 整页登录引导 → 搜号 → 选范围/格式 → 实时逐篇进度(可取消/单篇重试)。
- 缓存 session 有效期内免扫码,过期才弹回登录引导。

### M4 · 打包

- `electron-builder` 出未签名 `dmg`(arm64 + x64)+ `nsis`(x64)。
- 品牌「宝盒」图标,electron-builder 自动转 mac/win 各尺寸。
- 打包后真实启动 `.app` 验证(覆盖 undici external 已知陷阱)。

## 📦 下载

| 平台 | 架构 | 文件 |
|---|---|---|
| macOS | Apple Silicon (M1+) | `wx-kit-0.1.0-arm64.dmg` |
| macOS | Intel | `wx-kit-0.1.0.dmg` |
| Windows | x64 | `wx-kit Setup 0.1.0.exe` |

**macOS 放行**:右键应用 →「打开」→ 再次「打开」;或 `xattr -cr /Applications/wx-kit.app`。
**Windows 放行**:SmartScreen 提示→「更多信息」→「仍要运行」。
